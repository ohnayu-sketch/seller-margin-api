## 구현해야 할 전체 기능

### [A] Python 서버 (main.py) 추가 기능

#### A-1. 네이버쇼핑 시중가 조회 (기존 코드에 추가)
```
GET /search?query=상품명
→ 네이버쇼핑 API 호출
→ 반환값:
  {
    min_price: 최저가,
    avg_price: 평균가,
    max_price: 최고가,
    competitor_count: 경쟁 셀러 수,
    top_items: [
      { title, price, mall, review_count, rating, link, image }
    ]
  }
```

#### A-2. 리뷰/판매량 수집
```
GET /product-stats?query=상품명
→ 네이버쇼핑 상위 10개 상품의
  - 리뷰 수 합계
  - 평균 평점
  - 최근 30일 판매 추정량 (리뷰 증가율 기반)
  - 경쟁 강도 점수 (0~100)
```

#### A-3. 카테고리 자동 분류
```
GET /category?query=상품명
→ 네이버쇼핑 category1 필드 기반
→ 반환값:
  {
    category: "의류" | "식품" | "생활용품" | "전자기기" | "화장품" | "기타",
    sub_category: "상세분류",
    fee_rate: { 스마트: 6.6, 쿠팡: 8.0, 오픈: 15.0 },  // 카테고리별 실제 수수료
    risk_level: "낮음" | "보통" | "높음",  // 의류=높음(반품), 식품=높음(유통기한)
    special_notes: ["계절성 상품", "유통기한 주의"] 등
  }
```

### [B] 구글 Apps Script (기존 코드 확장)

#### B-1. 시트 구조 확장
기존 "상품목록" 시트에 컬럼 추가:
```
기존: ID | 상품명 | 원가 | 도매배송비 | 마켓배송비 | 마켓 | 수수료 | 판매가 | 수수료금액 | 순이익 | 마진율 | 저장일시 | 저장자

추가:
- 카테고리 (의류/식품 등)
- 경쟁강도 (낮음/보통/높음)
- 시중최저가
- 시중평균가
- 판매결정 (Y/N)
- 판매시작일
```

#### B-2. "판매기록" 시트 신규 생성
```
컬럼: 날짜 | 상품ID | 상품명 | 마켓 | 판매수량 | 판매가 | 매출 | 원가합계 | 순이익 | 마진율 | 저장자
```

#### B-3. "매입매출" 시트 신규 생성 (간이과세자용)

#### B-4. "월별통계" 시트 신규 생성 (자동 집계)

### [C] 프론트엔드 (seller-dashboard-v3.html 확장)

현재 탭 구조:
```
계산 | 상품목록 | 통계 | 설정
```

변경 후 탭 구조:
```
계산 | 소싱목록 | 판매관리 | 회계 | 통계 | 설정
```

#### C-1. 계산 탭 개선
```
기존 입력 필드 유지 +
- [신규] 상품명 입력 후 "시중가 조회" 버튼
  → Python 서버 호출
  → 시중 최저가/평균가 자동 표시
  → 카테고리 자동 분류
  → 경쟁강도 표시 (🟢낮음 / 🟡보통 / 🔴높음)
  → 리뷰 수 / 평균 평점 표시
- [신규] 카테고리 드롭다운 (자동 선택, 수동 변경 가능)
- [신규] "판매 결정" 토글 (소싱 → 판매 전환)
```

#### C-2. 소싱목록 탭 (기존 상품목록 개선)
```
기존 기능 유지 +
- 카테고리별 필터
- 경쟁강도 필터
- 판매결정 여부 필터
- 각 카드에 시중가 대비 우리 판매가 표시
- "판매 시작" 버튼 → 판매관리 탭으로 이동
```

#### C-3. 판매관리 탭 (신규)
```
상단: 판매 중인 상품 목록 (판매결정=Y인 것만)
각 상품 카드:
  - 상품명 / 마켓 / 등록 판매가
  - [판매 기록 추가] 버튼
    → 날짜 / 판매수량 / 실제 판매가 입력
    → 구글 시트 "판매기록" 시트에 저장
  - 누적 판매량 / 누적 매출 / 누적 순이익 표시
  - 판매 성과 그래프 (간단한 바 차트)

하단: 전체 판매 성과 요약
  - 이번 달 총매출 / 총순이익
  - 마켓별 매출 비중
  - 카테고리별 매출 비중
  - 베스트 상품 TOP3
```

#### C-4. 회계 탭 (신규, 간이과세자용)
```
상단 요약 카드:
  - 이번 달 매출 합계
  - 이번 달 매입 합계
  - 순이익
  - 부가세 예상액 (간이과세 1.5~4%)

거래 입력 폼:
  - 날짜
  - 구분: 매입 / 매출
  - 거래처
  - 품목
  - 금액
  - 증빙: 세금계산서 / 카드 / 현금영수증 / 기타
  - 메모
  → 구글 시트 "매입매출" 시트에 저장

거래 목록:
  - 월별 필터
  - 매입/매출 필터
  - CSV 내보내기 (세무사 제출용)

연간 부가세 계산기:
  - 간이과세자 기준 자동 계산
  - 1기(1~6월) / 2기(7~12월) 구분
  - 납부 예상액 표시
```

#### C-5. 통계 탭 개선
```
기존 통계 +
- 카테고리별 마진율 비교
- 월별 매출/이익 추이 차트
- 마켓별 판매 성과 비교
- 소싱 상품 중 판매 전환율
```

### [D] 구글 드라이브 자동 정리

Apps Script에서 구현:

```
📁 셀러마진 (루트 폴더)
├── 📁 2026년
│   ├── 📁 01월
│   │   ├── 📄 월별리포트_2026-01.pdf (자동 생성)
│   │   ├── 📄 매입매출_2026-01.csv (자동 내보내기)
│   │   └── 📁 영수증
│   ├── 📁 02월
│   └── ...
├── 📁 상품분석
│   └── 각 상품별 분석 자료 자동 저장
└── 📄 마스터시트 (구글 시트, 항상 최신)
```

Apps Script 트리거:
```
매월 1일 자정: 월별 리포트 자동 생성 + 드라이브 정리
매일 자정: 구글 시트 백업
```
