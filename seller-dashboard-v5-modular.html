가격 변동, 주문 알림, 재고 경고 등을 카카오톡 나에게 보내기로 자동 수신합니다. 가격 추적 사용 시 Apps Script 스크립트 속성에도 KAKAO_TOKEN을 등록해주세요.

발급 가이드:
1. developers.kakao.com 접속 → 앱 생성
2. 앱 키 → REST API 키 복사 → 위 필드에 입력
3. 카카오 로그인 활성화 → 동의항목에 카카오톡 메시지 전송 추가
4. 인가 코드 발급 → Access Token 발급 → 위 Token 필드에 입력
5. "테스트 알림 보내기" 클릭으로 정상 여부 확인
