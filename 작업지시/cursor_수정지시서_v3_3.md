## 수정 1 — 계산 탭: 시중가 조회 결과에 실상품 링크 카드 표시

### 현재 문제
최저/평균/최고가 숫자만 표시되고 어떤 상품인지 확인 불가.

### 수정 내용
시중가 조회 결과 아래에 실제 상품 카드 표시:

```
시중 최저가 19,800원 근처 상품:
┌─────────┬─────────┬─────────┐
│[이미지]  │[이미지]  │[이미지]  │
│19,800원 │21,500원 │23,000원 │
│쿠팡     │스마트스토어│G마켓   │
│[🔗 보기]│[🔗 보기]│[🔗 보기]│
└─────────┴─────────┴─────────┘
```

- 이미지 클릭 또는 [🔗 보기] 클릭 시 해당 판매 페이지 새 탭으로 열림
- top_items 데이터의 image, link, price, mall 필드 활용
- 이미지 로드 실패 시 기본 이미지 표시 (에러 무시)
- 최저가 근처 3개, 평균가 근처 3개 표시

## 수정 3 — 희망마진율 슬라이더 + 실시간 계산

### 현재 문제
마진율을 숫자로 직접 입력해야 함. 실시간 반영 안 됨.

### 수정 내용
```
희망 마진율
[─────●──────────] 15%
  5%            50%

숫자 입력란과 슬라이더 동시 표시:
- 슬라이더 움직이면 숫자 입력란 자동 업데이트
- 숫자 입력하면 슬라이더 위치 자동 업데이트
- 둘 다 변경 시 마진 계산 결과 실시간 반영

슬라이더 범위: 최소 1% / 최대 80% / 기본값 15%
눈금 표시: 10%, 20%, 30% 위치에 마크

input[type=range] 스타일:
- 트랙: #2a2d35
- 활성 트랙: #4ade80
- 핸들: #4ade80, 크기 24px (모바일 터치 쉽게)
```

## 수정 4 — 회계 탭 버그 수정 4가지

### 4-1. 카드 폰트 넘침 수정
```css
/* 매출/매입 카드 숫자 넘침 방지 */
.stat-card .amount {
  font-size: clamp(1rem, 4vw, 1.8rem);  /* 화면 너비에 따라 자동 조절 */
  word-break: break-all;
  overflow: hidden;
}
/* "0원 (연추정)" 같은 긴 텍스트도 줄바꿈 처리 */
.stat-card .label {
  font-size: 0.75rem;
  line-height: 1.3;
}
```

### 4-2. 서류 종류 버튼 토글 (선택 → 재클릭 시 해제)
```javascript
// 현재: 클릭 시 선택만 됨
// 변경: 이미 선택된 버튼 재클릭 시 선택 해제

function selectDocType(type) {
  if (selectedDocType === type) {
    selectedDocType = null;  // 해제
    // 모든 버튼 활성화 해제
  } else {
    selectedDocType = type;  // 선택
    // 해당 버튼만 활성화
  }
}
```

### 4-3. 손익계산 섹션 추가 (회계 탭)

- 판매기록 시트 + 매입매출 시트 데이터 합산
- 월별 필터 적용

### 4-4. 날짜 형식 수정
```javascript
// 현재: "2026-03년 3월" (잘못된 형식)
// 수정: "2026년 3월" 또는 "2026-03"

// 월 필터 드롭다운 표시 형식 변경
function formatMonthLabel(yearMonth) {
  // yearMonth: "2026-03"
  const [year, month] = yearMonth.split('-');
  return `${year}년 ${parseInt(month)}월`;
  // 결과: "2026년 3월"
}
```

## 수정 5 — 소싱 탭 숫자 "0" 제거

### 현재 문제
소싱 탭 제목 아래 "0"이 표시됨.

### 수정 내용
소싱목록 탭 배지(badge) 관련 코드 확인:
```javascript
// 상품 수 배지가 0일 때 숨기기
// 현재: 항상 표시
// 변경: 0이면 숨김, 1 이상이면 표시

if (count > 0) {
  badge.textContent = count;
  badge.style.display = 'inline';
} else {
  badge.style.display = 'none';
}
```
탭 버튼 아래 또는 탭 텍스트 아래에 별도로 "0"이 렌더링되는 요소가 있으면 제거.
